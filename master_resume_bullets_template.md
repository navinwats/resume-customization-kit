# Master Resume Bullets

This is your single source of truth. Every bullet you might ever use on a resume lives here, organized by role. Pull from this file when customizing — never invent bullets that aren't here.

**Rules for this file:**
- Only include bullets backed by real, verified outcomes
- Include every metric you can verify — percentages, dollar amounts, scale, benchmarks beaten
- Write bullets in past tense (except your current role)
- Keep bullet headers consistent: `**Header:** Body text.`

*Tips for building this file:*
- *Go role by role through your LinkedIn and write 8–12 bullets per role*
- *For each bullet, ask: "What did I do, how did I do it, and what was the result?"*
- *Look up old performance reviews, project docs, and email threads to find real numbers*
- *If you don't have a metric, find a scale marker (team size, budget, number of countries, user base)*
